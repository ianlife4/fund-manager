      📊 匯入歷史週績效

      從 Excel 複製週績效表格貼到下方。格式：每行一週，欄位用 Tab 分隔。
      必要欄位：日期、銀行現金、證券市值、總淨值，可選：本週入/出金、備註
      範例：1月18日 <tab> 56505 <tab> 3018893 <tab> 3075398 <tab> 0 <tab> 備註

      使用說明

      📋 持倉管理：每週更新「現價」即可自動計算損益、獲利率、市值佔比。

      💰 出入金：記錄每筆入金/出金，系統自動計算基金總市值（市值 + 現金）。

      📈 績效曲線：點選總覽頁「記錄本週」即可儲存當週快照，績效圖自動更新。

      🥧 配置分析：自動依市值計算各股及產業別佔比。

      💾 資料儲存：所有資料存於瀏覽器本地，建議定期匯出 JSON 備份。
